# DESIGN_DECISIONS.md

本檔只記錄使用者已確認的設計決策。

不要把猜測、建議、未確認方案寫進來。

## 已確認

- 專案方向是「猜數字 + 自動出兵攻城」的混合玩法。
- 後續會加入點數、技能、士氣、節點防禦建築、菁英單位、王單位。
- 文件與溝通使用繁體中文。
- `AGENTS.md` 應作為 agent 行為守則，而不是一般補充資料。
- `.agent_memory/` 可作為 agent 記憶資料夾。
- 只有在有修改檔案時，才需要更新 `WORK_LOG.md`。
- 每次修改檔案後，必須依情況更新 `.agent_memory/` 中的相關記憶檔案。
- 每次開始工作時固定必讀 `AGENTS.md`；其他 `.agent_memory/` 檔案由任務內容、風險、使用者或系統需求判斷是否讀取，不必每次強制讀完全部。
