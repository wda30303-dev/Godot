# PROJECT_CONTEXT.md

本檔記錄專案目前穩定事實。

只有在專案狀態確定改變時才更新。不要把猜測、尚未確認的設計或臨時想法寫進本檔。

## 專案類型

Godot 4.6 2D 策略戰鬥原型。

核心是：

- 猜數字
- 自動出兵
- 雙方攻城
- 戰場節點佔領
- 未來擴充技能、點數、士氣、防禦建築、菁英單位、王單位

## 目前主場景

- `game_root.tscn`

## 目前主要腳本

- `guess_engine.gd`：猜數字答案生成與 A/B 判定。
- `input_panel.gd`：玩家輸入與送出。
- `game_controller.gd`：遊戲主要流程、出兵、猜測結果、UI 更新。
- `ControlNode.gd`：戰場節點佔領。
- `PlayerCastle.gd`：玩家城堡 HP 與受傷。
- `EnemyCastle.gd`：敵方城堡 HP 與受傷。
- `Units/`：士兵、敵方士兵、菁英、特殊單位、王單位相關內容。

## 目前玩法狀態

玩家目前猜 3 位不重複數字。

達成 `3A` 時視為猜中，會重新產生答案、清除玩家猜測紀錄、增加特殊能量。

玩家與敵人會自動生成士兵。士兵會往敵方城堡方向移動，遇到敵人會攻擊，抵達城堡偵測範圍後會造成城堡傷害。

戰場中間目前有 5 個控制節點。士兵停在節點中心時會佔領節點。
